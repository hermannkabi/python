Kirjuta programm, mis loob järgnevad muutujad:

Muutuja nimi Andmetüüp Kirjeldus
nimi str Küsi see muutuja kasutaja käest (kasuta input funktsiooni)
vanus int Siia pane enda vanus
pikkus float Pane selleks enda pikkus meetrites (ujukomaarvuna)
hobid string Mõtle paar hobi enda kohta (nt lugemine)

Prindi välja järgmine lause: NIMI on VANUS-aastane PIKKUS m pikk inimene, kellele meeldib HOBID
